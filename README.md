# Procrastination Prediction using Machine Learning

This project analyzes student procrastination behavior and its impact on academic performance using machine learning models.

Dataset:
Student survey dataset from Kaggle with 451 responses.

Models Used:
- Logistic Regression
- Decision Tree
- XGBoost

Results:
The XGBoost model achieved around 85.6% accuracy in predicting academic performance based on procrastination behavior.

Tools Used:
Python, Google Colab, Scikit-learn, XGBoost
